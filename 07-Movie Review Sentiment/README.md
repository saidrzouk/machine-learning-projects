# Movie Review Sentiment Classifier

A machine learning project for binary sentiment analysis on movie reviews.

## Project Structure

- `data/` - dataset files
- `src/` - preprocessing and training scripts
- `models/` - saved trained model artifacts
- `results/` - evaluation outputs
- `app.py` - Streamlit demo app
- `Movie_review_sentiment.ipynb` - notebook for exploration and experiments

## Run

Train your model from `src/`, then launch:

```bash
streamlit run app.py
```
