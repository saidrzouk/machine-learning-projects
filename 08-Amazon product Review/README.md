# Amazon Product Review Classifier

A machine learning project for sentiment classification of Amazon product reviews.

## Project Structure

- `data/` - dataset files
- `src/` - preprocessing and training scripts
- `models/` - saved trained model artifacts
- `results/` - evaluation outputs
- `app.py` - Streamlit demo app
- `Amazon_Product_Reviews.ipynb` - notebook for exploration and experiments

## Run

Train your model from `src/`, then launch:

```bash
streamlit run app.py
```
